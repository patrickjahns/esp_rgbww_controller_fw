ESP RGBWW Firmware

This repository hosts the firmware for the open hardware controller esp_rgbww

Link for OTA update http://patrickjahns.github.io/esp_rgbww_firmware/release/version.json

Link to seperate files

  • Bootloader (rboot)
  • Rom
  • Fileystem

Filesystem includes the latest version of web interface

Use esptool2 for flashing

Example command:

esptool -p COM3 -b 115200 write_flash -ff 40m -fm qio -fs 32m 0x00000 rboot.bin 0x02000 rom0.bin 0x300000 spiff_rom.bin
